üêõ Troubleshooting

  • Nothing happens: The protocol might not be registered. Try building and running Clipify first.
  • Browser blocks the link: Some browsers block custom protocols. Try copying the URL to terminal instead.
  • App doesn't receive the link: Check the Deep Link Tester component in the app for debugging info.
  • Authentication fails: The test tokens are fake - real authentication requires valid JWT tokens from your server.
